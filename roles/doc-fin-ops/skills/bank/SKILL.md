# /bank

## Что делает

Берёт банковскую выписку Точки (CSV или текстом) и разносит строки в `fin.economics` OzmaDB: правильные категории, контрагенты, теги, пометка `notification_who = "Клодик"`.

## Как работает

1. Загружает актуальный мастер-промпт из Notion: page `320612c762af81ceb53bcfc18fd27514`.
2. Если выписка ещё не приложена — просит её у пользователя (файл / текст / ссылка).
3. Действует строго по мастер-промпту. Перед массовой вставкой — показывает превью первых 3–5 строк на проверку.
4. После вставки — отчитывается: сколько записей внесено, какие категории, общие суммы.

## Важно

- Если строка похожа на Tinkoff-эквайринг — разбивает на **4 записи** (доход / зеркальный расход / комиссия / НДС). См. [systems.md §4.2](../../../../shared/knowledge/systems.md).
- Удаление записей: сначала чистить зависимости в `fin.pl_report_final`, потом `fin.economics`.
- Не править вручную — править мастер-промпт в Notion, чтобы изменения попали ко всем.
