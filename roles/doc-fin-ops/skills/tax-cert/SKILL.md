# /tax-cert

## Что делает

По PDF-справке (или по ФИО + году) сверяет с базой Ozma:

1. Находит физлицо в `base.people` по ФИО + паспорту/ДР (не по ИНН — поле `itn` часто пустое).
2. Сверяет паспорт, дату рождения, дату выдачи паспорта.
3. Считает сумму расходов из `fin.transactions` по правилам справки (см. мастер-промт).
4. Сравнивает с суммой в справке, показывает построчно включённые и исключённые транзакции с причинами.
5. Ловит флаги: преждевременная справка (программа ещё не окончена), опечатки в шаблоне, расхождения по физлицу.

## Как работает

1. Загружает мастер-промт из Notion: page `372612c762af811f9adcccb8661617ed` («🤖 Промт: Проверка справки об оплате обучения»).
2. Просит у пользователя PDF справки или связку (ФИО, год). Если PDF — извлекает данные.
3. Действует строго по мастер-промту.
4. Возвращает отчёт в формате: физлицо ✓/⚠️, сумма ✓/расхождение, таблицы «включено/исключено», ссылки на сомнительные транзакции.

## Ключевые фильтры суммы (кратко — для контекста, полный список в Notion)

**Включаем:** `tks_state IN ('CONFIRMED', 'AUTHORIZED')`, `account_to=>contractor=>is_our_organization = true`, `is_deleted=false`, `is_plan=false`, `customer=>id = <person_id>`, год транзакции.

**Исключаем:**
- `tks_state = 'Ожидается оплата'` (НИКОГДА не учитывается — общий принцип из [shared/CLAUDE.md](../../../../shared/CLAUDE.md))
- оплаты сертификатами (`account_from=>type = 'Сертификат'`)
- переводы в сертификат (`request=>cancellation_status = 'Перевод в сертификат'`)
- полные возвраты и отмены
- покупки сертификатов
- оплаты от компаний (B2B)
- неиспользованные абонементы
- проценты по рассрочке (только тело программы)
- оплаты до 2025-01-01 (вычет распространяется с 2025; лабы/интенсивы зимы 2025 — по запросу, отдельной справкой)

## Важно

- Справку выдают **после окончания обучения** — иначе риск возврата → уточнённая декларация. Если программа ещё идёт — флаг ⚠️.
- В шаблоне справки в Notion есть опечатка «отв**е**ственностью» вместо «ответственностью» — проверять при печати.
- Подписант — Екатерина Гусакова, дата = дата получения запроса (для метчинга с нумерацией).
- Скилл **только проверяет**, не правит. Правки вносит «Менеджер по документообороту».
