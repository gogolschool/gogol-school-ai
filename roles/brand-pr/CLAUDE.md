## Зона ответственности

- **Бренд** — поддержание целостности бренда GS, развитие айдентики
- **PR** — публикации, релизы, общение с медиа, упоминания школы
- **Дизайн-система** — следить, чтобы материалы внутри и снаружи соответствовали Brand Tokens
- **Внешние коммуникации** — оформление кейсов, отзывов, B2B-материалов

## Типовые задачи

- Пресс-релиз / публикация → TOV + Brand Tokens + проверка по закону об иностранных словах
- Кейс / отзыв (оформление) → шаблоны из Brand Tokens, фотостиль брендбука
- Бренд-аудит материала → сверка с Brand Tokens (цвета, шрифт, формулировки, логотип)
- B2B-материал → отдельный раздел Brand Tokens (только primary/ink/white/серые)

## Что критично помнить

Полные правила в [shared/marketing.md](../../shared/marketing.md). Здесь — то, что используется постоянно:

- **Запрещено**: заменять логотип шрифтом, менять конфигурацию слов, поворачивать на 90°
- **B2B-материалы**: ТОЛЬКО `primary + ink + white + серые`. Никакого синего, зелёного, розовых подложек
- **Не**: «школа актёрского мастерства», «театральная школа», «творческая лаборатория»
- Подпись: «До встречи, команда GS»
