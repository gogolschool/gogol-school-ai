> Общий контекст про Gogol School, тон, бренд, закон об иностранных словах — в [shared/CLAUDE.md](../../shared/CLAUDE.md). Этот файл — только про специфику роли.

## Зона ответственности

Маркетинговые задачи Gogol School:

- Сегментация контактов, ведение маркетинговых списков (`marketing.lists`, `marketing.campaigns`)
- Подготовка и запуск email-кампаний (через `unisender`)
- Подготовка анонсов, рассылок, описаний программ
- Работа с гипотезами по методу ICE
- Конкурентный анализ

## С какими системами работаю

- **OzmaDB**: схема `marketing` (кампании, списки, реферальные промокоды), `crm.actions_for_contacts` (для сегментации)
- **Notion**: маркетинговые документы, бренд, TOV, prompt library
- **Unisender**: email-рассылки
- **Google Sheets / Docs**: планы, отчёты, креативы
